## 🍩 Donut Order

Django を使用して開発するドーナツ通販システムです。  
商品の管理、在庫、ユーザー注文機能などを備えた実務を意識した学習用プロジェクトです。

---

## 🔧 使用技術

- Python 3.13
- Django 5.x
- SQLite (開発環境)
- HTML / CSS / Bootstrap（予定）

---

## 📦 機能一覧（予定含む）

| 機能 | 詳細 |
|---|---|
| 商品登録/編集/削除 | 管理者がドーナツ商品を管理 |
| カテゴリ管理 | 商品カテゴリを作成・紐付け |
| 在庫管理 | 在庫数の更新と確認 |
| 顧客情報管理（予定） | ユーザー登録・編集 |
| 注文機能（予定） | カートに追加・購入処理 |
| 管理画面 | Django admin による管理ビュー |
